# Stock Portfolio Analytics Backend

## Overview

A Nest.js + Fastify backend providing:

- Health check (`GET /health`)
- API reachability probe (`GET /api-check`)

## Tech Stack

- Node.js 18, Nest.js, Fastify
- Docker, AWS Fargate
- GitHub Actions for CI/CD

## Quickstart

1. Clone repo

```bash
$ git clone git@github.com:you/portfolio-backend.git
$ cd portfolio-backend
```

2. Install dependencies

```bash
$ npm ci
```

3. Run locally

```bash
$ npm run start:dev
# listens on http://localhost:3000
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Docker

Build and run:

```bash
$ docker build -t portfolio-backend .
$ docker run -p 3000:3000 portfolio-backend
```

## CI/CD

On each push to main, GitHub Actions will:

Lint & test
Build & push Docker image to ECR
Trigger a new deployment on AWS Fargate
